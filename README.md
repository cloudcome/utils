# A new Vue App - utils

## start
```shell
npm install

# Synchronize the Openapi documentation to the local project.
npm run openapi-axios
```

## dev
```shell
npm run dev
```

## lint and fix
```shell
npm run lint
# or
npm run lint:fix
```

## release
```shell
npm run release
```

## build
```shell
npm run build
npm run preview
```
